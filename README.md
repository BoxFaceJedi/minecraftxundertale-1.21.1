Minecraft x Undertale

A Minecraft mod created for the self-directed learning exhibition 2025 at 惠文高中.
This is a Undertale themed combat overhual mod for Minecraft 1.21.1/1.20.1
features include: 
  custom key detection (src/main/java/net/mxumod/mxumod/util/Keybinding.java)
  toggleable combat mode (src/main/java/net/mxumod/mxumod/event/EnterCombatmode.java)
  custom abilities (src/main/java/net/mxumod/mxumod/skill)
  custom entities (src/main/java/net/mxumod/mxumod/entities)
